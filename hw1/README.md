# Resources and implementation of Project I - Random Forests.

### Folder structure:
- `instructions.md` contains the instructions for the task.
- `hw_tree.py` contains the implementation of the task.
- `Project I report` contains the report.
- `*.tab` contain the data.
- `imp_analysis.ipynb` contains some code snippets used for visualizations.
- `test_hw_tree.py` contains some basic tests for the tree and forest implementations.

### Running the code:
- Create a virtual environment: `uv venv`
- Activate: `.venv\Scripts\activate`
- Install necessary dependencies: `uv pip install -r requirements.txt`
- Run the script: `python hw_tree.py`
